    Request body

    Before you can post data you need to get the below data which is requied for crating the POST request.

        "account_code=" // you need to get this from the v1/banks 
    "&account_type=" // for VND only banks are supported
    "&api_token=" // API Key 
    "&bic_swift=" // SWIFT ID for the bank
    "&client_txn_id=" // Your internal transaction or transaction identifier 
    "&deposit_number=" // Account information 
    "&destination_currency="  // vnd 
    "&name=" // Name of the beneficiary 
    "&recipient_address=" // Address information. 
    "&recipient_city=" // City 
    "&recipient_city_of_birth=" // City of birth
    "&recipient_country=" // Two character country code
    "&recipient_country_of_birth=" 
    "&recipient_country_sub_division=" // State 
    "&recipient_date_of_birth=" Date of birth in YYYY-MM-DD format!!! 
    "&recipient_district=" // district 
    "&recipient_email=" // email address 
    "&recipient_phone_number=" // Country code followed by mobile number !!! do not include the + or 00 before the country code. 
    "&recipient_postal_code=" // Zip code / postal code. 
    "&recipient_province_of_birth=" 
    "&recipient_street="
    "&recipient_town_name="
    "&sender_address="
    "&sender_city="
    "&sender_country=" 
    "&sender_country_of_birth="
    "&sender_date_of_birth="
    "&sender_email="
    "&sender_first_name="
    "&sender_gender="   
    "&sender_id_number=" 
    "&sender_id_type="
    "&sender_last_name=" +
    "&sender_mobile_phone_country=" // Two character country code
    "&sender_mobile_phone_number="
    "&sender_street=" 
    "&sender_type=Individual" // Has to be individual 
    "&sender_zipcode=" +
    "&source_amount=" // USD value to be credited
    "&source_currency=usd"
    "&tonce=" // epoch time.
